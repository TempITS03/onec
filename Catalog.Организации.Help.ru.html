Справочник «Организации»

Справочник «Организации» предназначен для ведения списка собственных организаций,  входящих в состав предприятия (группы), и хранения постоянных сведений о них или нескольких организаций если это удаленный программист или администратор. В таком случае у удаленного сотрудника собственная база, а информация о "заданиях" выполняется с помощью обмена данными.

Имеет следующие реквизиты:

  • Префикс – хранится префикс, который присваивается при нумерации документов данной организации;
  • ИНН – идентификационный номер налогоплательщика
  • КПП – код причины постановки на учет (для РФ)
  • Наименование полное – для размещения полного официального наименования организации
  • Код по ОКПО (для РФ)
  • Код по ОКАТО (для РФ)
  • Код ИМНС – код налогового органа, где организация состоит на учете (для РФ)
  • Юр. / физ. Лицо – указывается, является ли организация юридическим лицом, или нет
  • Вести взаиморасчеты с окладниками - если данная организация ведет расчеты с сотрудниками по окладам
  • Основная валюта взаиморасчетов - валюта в которой организация ведет взаиморасчеты с заказчиками.

Важно. "Основная валюта взаиморасчетов" - обязательно для заполнения. Для этого следует сперва заполнить справочник "Валюты".
