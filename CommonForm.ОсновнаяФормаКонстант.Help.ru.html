Основные параметры системы

"Обмен данными"

  • "Каталог обмена данными" - ссылка на файловый ресурс куда будут сохранены файлы обмена
  • "Использовать обмен данными через почту" - при  взведенном флажке система будет использовать прием и отправку через почтовый ресурс
  • "Сжимать файл обмена" - при взведенном флаге будет происходит сжатие файла обмена, при этом оригинальный файл не будет удален
