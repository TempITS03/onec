Список премий по сотрудникам организаций. Для начисления дополнительных премий по сотрудникам есть механизм начисления премий от выроботки. Данные для расчета премий берутся из этого регистра. Поскольку удаленные программисты или администраторы могут сотрудничать не с одной организацией или не для всех сотрудников небходимо наложить премии, то есть смысл использовать детализацию по измерениям "Организация" и "Сотрудник".

Правила заполнения данных.

Необходимо заполнить все поля и если для сотрудника предусмотрена не однократная премия, а разная от зависимости отработанных часов, то необходимо указать такие шкалы. Например:

Количество часов 100  Процент 1%

Количество часов 150 Процент 3%

Документом расчета премии выступает "Дополнительные начисления".
