Настройка обмена данными задачами

Для начала необходимо указать в "Основных параметрах системы" вид обмена данными. Данную форму можно вызвать из "Настройка и администрирование" - "Сервис" - "Основные параметры системы" 

В списке обмена данными необходимо указать центральный узел и задать этому узлу "Код" и ""Наименование". Для перефирийных узлов тоже необходимо задать "Код" и "Наименование".

Настройка перефирийных узлов

Если обмен данными происходит с несколькими перефирийними узлами или же используется обмен по почте, то необходимо внести более точные настройки для обмена. При обмене по почте необходиме в элементе узла указать e-mail адрес куда отправлять почту.

При использовании выгрузки нескольких задач на разные узлы, необходимо в узлах заполнить фильтр обмена, для этого необходимо взвести флаг "Использовать фильтр обмена" и конкретизировать фильтр для узла.

"Параметр фильтра" - содержит два значения "Исполнитель" и "Куратор", при этом "Значение фильтра" содержит ссылку на справочник сотрудников. Что же происходит при указании данных фильтров:

система, когда делает выгрузку данных, а алгоритм выгрузки состоит из обхода соответствия регистрации документа к узлу, сверяет текущий документ с указанным фильтром и если данные не совпали, то документ с регистрации узла удаляется, в противном случае создается файл с выгрузкой задания.

Важно. Настройка фильтра обмена необходима для того, чтобы не отправить задачу одного пользователя соврешенно другому. Фильтр стоит использовать, когда обмен происходит с несколькими узлами. При загрузке, фильтр обмена на загрузку данных не влияет.
