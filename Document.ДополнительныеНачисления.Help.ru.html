Документ «Дополнительные начисления»

Для отражения в учетной системе дополнительных начислений таких как: "Премия", "Бонусы", "Курсовая разница" или других начислений, необходимо использовать данный документ.

Механизмы заполнения данными:

Под механизмами заполнения подразумевается упрощение работы пользователя по заполнению данными табличных частей документа. На командной панели табличной части "Начисления" есть подменю заполнить:

  • "Заполнить сотрудниками организации" - при выборе данной комманды система автоматически заполнит табличную часть сотрудниками выбранной организации с учетом регистра сведений "шкала тарифов". Будут взяты только те сотрудники с которыми организация ведет расчет по даному регистру. При заполнении автоматически будет подставлено вид начисления "премия" как предопределенный элемент справочника "Перечень дополнительных начислений", суммы необходимо указать вручную для каждого сотрудника.
  • "Начислить премию окладникам" - при расчете премий окладникам, следует учитать момент анализа количества отработанных часов, то есть документ желательно вводит в конце месяца так сказать по итогам месяца, поскольку количество затраченных часов берется как движение приход за период от начала месяца даты документа и по конец дня даты документа. При начислении система анализирует фактические трудозатраты количества часов из регистра накопления "РасчетыССотрудниками" по ресурсу "КоличествоЧасов" и сравнивает на возможность того, что сотруднику принадлежит выплатить премию, как разница времени между количеством фактических часов и количеством обязательных часов, которые задаются в регистре сведений "шкала тарифов". Если данная разница часов больше нуля то она принадлежит к расчету воторого этапа сравнивания. Сравниваются количество фактически затраченых часов из количеством часов, которые указаны в регистра сведений "шкала премий". И если количество часов принадлежит какой-то из границ то берется самая максимальная граница к которой принадлежит даное количество часов и берется значение ресурса "процент" по которому вычисляется сумма премии. При заполнении автоматически будет подставлено вид начисления "премия" как предопределенный элемент справочника "Перечень дополнительных начислений".
СуммаПремии = Оклад*Процент;
  • Ручное заполнение - вручную выборка всех полей и их заполнения.

При проведения документа делается движение приход регистра накопления "РасчетыССотрудниками" на сумму, количество часов остается не изменным.
